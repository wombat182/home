# Dønna

Statisk nettside for Dønna på GitHub Pages.

## Struktur

- `index.html` — forside
- `opplev/` — artikler, severdigheter og historie
- `turruter/` — fjell, sykkel, padling og ruter
- `lei/` — utleie og fremtidig lokal markedsplass
- `bedrifter/` — lokale aktører og kategorier
- `planlegg/` — transport og praktisk info
- `artikler/` — artikkelsider
- `flyer-sup.html` — hvit A5-flyer for SUP-utleie

Ingen `CNAME` er inkludert. Siden peker derfor ikke til eget domene ennå.
